## Review Once

```dataview
TABLE file.mtime as "Today" FROM "" OR "Journal" WHERE file.mtime >= date(today) AND file.mtime < date(tomorrow) SORT file.mtime desc
```





```dataview
TABLE file.mtime as "Last Week" FROM "" OR "Journal" WHERE date(file.mtime) >= date(today) - dur(7 days) SORT file.mtime desc
```
