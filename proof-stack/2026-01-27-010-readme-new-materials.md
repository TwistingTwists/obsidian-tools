### 3. 2026-01-27-007-error-analysis-framework.md
**Purpose:** Spreadsheet-based method to prove reliability with data
**Contains:**
- Error taxonomy (5 categories you'll track)
- Google Sheet structure (what columns to track)
- Quarterly error analysis report template
- How to use error data in sales conversations
- Metrics: Detection Rate, Recovery Rate, Impact Rate, MTTR
- Why this proves reliability (vs. just claims)

**How to use:**
- Create a Google Sheet with the column structure
- Log errors weekly (at minimum)
- Calculate quarterly metrics
- Use error data in sales conversations: "We processed 150K requests, detected 95% of failures automatically"
- Show quarterly trends: Error rate should go down, detection should go up

**Status:** Framework ready (start collecting data immediately)
