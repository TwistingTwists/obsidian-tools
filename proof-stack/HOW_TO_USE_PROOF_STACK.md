## Keys to Success
1. **Speed:** Reply within 5 minutes with proof, not promises
2. **Authenticity:** Show real work, not polished mockups
3. **Tailoring:** Match proof type to buyer stage
4. **Simplicity:** Make it easy to access proof (one link)
5. **Consistency:** Update proofs quarterly as you ship new projects
